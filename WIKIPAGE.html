Creating an Interactive Stacked Bar Chart using Charts.js

How Risky Is Your Team's 1st Round Draft Pick?

What the tool is and why it is used.

For this assignment I created an interactive stacked bar chart using Charts.js. This is a website that provides libraries and documentation that allows you to easily create interactive charts.

What is it Thinking?

This program reacts completely based on if/then processes. If you click on a pick number, then the overlay displays the pick information. If you click on the toggle switch, then the list changes its order.

Patterns, Trends and Comparisons

This interactive does not have a lot of analytical patterns or trends. But it allows you to change the pattern on the screen. You can switch between seeing the draft order, and seeing the picks in order based on how risky they are. You can compare different picks based on which ones are risky and which ones are safe. And within the picks, you can compare the best and worst players at those picks.

Labels, Legends and Keys

There are many labels on this program. First, risk meter that shows what colors are safe (green) and what colors are risky (red). Then the numbers to represent the different picks and allows you to look in more detail into the different picks. Within the pick profiles are the Best and Worst players drafted at that pick.

Story Enhancement

This interactive enhances the readers experience by allowing you easily move through different picks to discover which ones are more risky than other. It is easy to read, use and understand.

Important Context

This interactive shows a variety of different ways to view the best and worst draft picks. Two easy and interesting way to look at the set of data. Plus a little more information about each pick.

Context Not Included

The only thing that this interactive doesn't include that I think it should is a list of notable safe picks and notable risky picks.
