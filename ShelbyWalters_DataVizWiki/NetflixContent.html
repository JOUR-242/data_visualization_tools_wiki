What is GeoCharts?

GeoCharts is a very simple and easy to use data visualization tool. GeoCharts is map of a country, continent or region with areas identified in one of three ways:

  • Regions- colors who regions, such as countries, provinces or states
  • Markers- uses circles to designate regions that are scaled according to a value that you specify
  • Text- labels the regions with identifiers, such as 'Russia' or 'Asia'

Where to get GeoCharts

You can find the tool on Google Developers. Google has several different visualization tools, but I chose GeoCharts. Geocharts does not need any additional software. All the code that you need is provided to you, so you just have to add your own data. You can take the code and edit it in a text environment like JSFiddle.

How GeoCharts Works

Before using GeoCharts, I used Google Spreadsheets to scrape my data from Finder.com, which had the number of Netflix TV shows and movies available in each country.

I had to organize my data before I could put it into my GeoChart. I alphabetized the countries and used the sum formula to add the number of movies and TV shows.

GeoCharts uses Javascript and HTML code

The map is created by using a function that draws the type of GeoChart that you want, in this case regions.

After the computer knows what it is making, you specify and load your data into arrarys.

The options variable allows you to change how your GeoChart looks. You can change the colors represented on your map and values of the axis.

Finally, the variable chart, which is the map, is drawn using your data and options you included.

After you finish your Javascript code you have to link your Javascript to your HTML file. The part in green 'netflix_content.js' is the name of the Javascript file that you are linking to.

Your final Geochart should look something like this.

Limitations

GeoCharts does not allow you to zoom in and out or drag around the map. I also found that because the tool is very simple you can only put in one column of numbers; it only measures/compares the first number you put in although you can see all the other numbers when you hover over the country.
